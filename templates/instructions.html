1. Пояснення вкладок сайту

Browse

Вкладка Browse служить для переходу на стартову сторінку, де користувач може переглянути весь список конструкцій українського конструктикону. На сторінці у вікні пошуку ‘Search in names and illustrations’ користувач може задати пошук будь-яких елементів, що входять до складу конструкції, наприклад слово, словоформи, категорії NP, Acc, VP, та інші. У списку відображаються номер, узагальнена формула (ім’я) та ілюстрація конструкцій. Натиснувши на будь-яку конструкцію можна відкрити її опис. Кнопка "Show additional information" під описом конструкції дозволяє побачити додаткову інформацію про конструкцію.

Daily Dose

Вкладка Денна порція (Daily Dose) дозволяє створювати випадкову добірку із п'яти конструкцій, які можна відсортувати залежно від рівня володіння мовою (CEFR - Загальноєвропейські компетенції володіння іноземною мовою), який користувач може налаштувати у відповідному меню. На вкладці відображається номер, ім'я та ілюстрацію конструкції. Для отримання повного опису необхідно натиснути на відповідну конструкцію.

Advanced Search

Вкладка Розширений пошук (Advanced Search) служить для швидкого пошуку конструкцій ресурсу за різними параметрами, які може встановити користувач у відповідних полях.

Statistics

Модуль Statistics відкриває статистичні данні та дозволяє систематизувати всі конструкції згідно з їх синтаксичного та семантичного типу, синтаксичної функції незмінного елемента та інших параметрів.

Instructions

На сторінці Інструкції користувач може завантажити пояснення українською та англійською мовами щодо архітектури сайту, скорочень та термінів, що використовуються у ресурсі.

About

Вкладка Про проект (About) містить інформацію про ресурс, код і джерело даних, розробників, та публікації.

1. Explanation of tabs

Browse

The Browse button opens the start page, where the user can view and scroll through the entire list of constructions of the Ukrainian Constructicon. The search window 'Search in names and illustrations', helps the user search for any elements of the construction, such as a word, word form, categories like NP, Acc, and others. For each construction displayed in the list there is an ID number, a Name and an Illustration of how it is used. By clicking on any construction, the user will get the description of the construction. The button "Show additional information" under the construction description enables the user to see additional information about the construction.

Daily Dose

The Daily Dose page offers the user five constructions chosen at random and sorted by language proficiency level (CEFR that stands for Common European Framework of Reference), which the user can choose from the corresponding menu. The ID number, the Name and the Illustration of each construction are displayed in the list. To get the full constructicon entry, the user should click on the corresponding construction.

Advanced Search

The Advanced Search tab is used to search and filter the constructions in the resource according to various parameters that the user can set in the corresponding fields.

Statistics

The Statistics page displays statistical data and makes it possible for the user to systematize all constructions according to their syntactic and semantic type, syntactic function of the fixed lexical part (anchor), and other parameters.

Instructions

On the Instructions page the user can find explanations of the website architecture, abbreviations, and explanations of the terms used in the resource, provided both in Ukrainian and English.

About

The About page contains information about the resource, the code and data source, the developers, and dissemination.

2. Abbreviations and symbols

Each construction is represented by its name and a short illustration: e.g. найти-Pst NP-Acc! Нашли развлечение! (‘What a bad way to amuse yourself! [lit. found amusement]’). The name of a construction is a short morphosyntactic formula that usually includes fixed lexical parts (called the anchor) as well as grammatical slots indicated by means of common abbreviations like NP, VP, etc. When necessary, we specify morphological characteristics of the fixed lexeme or a slot, where we use abbreviations according the Leipzig Glossing rules.
